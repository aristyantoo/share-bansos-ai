<div class="lang-en">
Cursor is an AI-first IDE that replaces VS Code with advanced agentic workflows. By running models locally or in the cloud alongside your project index, Cursor enables seamless code refactoring, bug-hunting, and feature implementation.

### Key Features
- **Composer Mode**: Generates multiple file changes concurrently and coordinates edits across your workspace.
- **Codebase Indexing**: Indexes your files locally so you can reference your whole project using `@codebase`.
- **Autocomplete (Tab)**: Suggests inline completions, edits, and cursor jumps ahead of your typing.

### How to Get Started
1. Download [Cursor](https://www.cursor.com) and import your VS Code configuration/extensions in one click.
2. Open your project folder.
3. Press `Ctrl + K` (Windows) for inline edit or `Ctrl + L` for codebase-aware assistant chat.
