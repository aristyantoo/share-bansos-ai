Kiro

by Kiro (Amazon / AWS) · AI-powered IDE
Visit Kiro

Kiro is an AI-powered IDE built on Code OSS that takes a spec-driven approach to development. Instead of jumping straight to code, Kiro guides you through writing structured specifications, design requirements, and task checklists — giving AI agents the context they need to produce reliable, maintainable code.

Starting Price Free $20/mo for Pro
Platform Desktop Win · Mac · Linux
Based on VS Code Code OSS
AI Models Bedrock AWS-powered
Key Features
📋
Spec-Driven Development
Define requirements, designs, and task checklists before writing a single line of code. AI agents follow your specs, not guesses.
🤖
Agentic Workflows
Autonomous agents analyze your codebase, write files, run tests, and manage multi-file changes — all in the background.
🔌
VS Code Ecosystem
Full compatibility with your existing VS Code extensions, keyboard shortcuts, and configurations. Zero migration friction.
☁️
Cloud-Agnostic
Run locally or in the cloud. No personal AWS account required — foundation models are served via Amazon Bedrock.
🔒
Built-in Security Hooks
Security scanning and compliance checks baked into the workflow so issues are caught at spec time, not after deployment.
📊
Multi-Agent Coordination
Orchestrate multiple specialized agents on different tasks simultaneously, with conflict detection and merge management.

How to Get Started
1
Download Kiro
Head to kiro.dev and download the installer for Windows, macOS, or Linux. Installation takes under a minute.
2
Sign in
Authenticate using Google, GitHub, or AWS SSO. Your existing VS Code profile can be imported automatically.
3
Open a project & write your first spec
Open any folder and use the Spec Panel to define what you want to build. Kiro will generate a task checklist and start executing.
⌘ K → "New Spec" to open the spec editor
